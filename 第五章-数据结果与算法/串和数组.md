# 串
字符串是一串文字及符号的简称，是一种线性表。其元素是字符。计算机中非数值处理的对象常常
是字符串。

## 串的定义及基本运算(串的抽象数据类型)
```
ADT String {
    数据对象:
        Data
    数据关系:
        Relation
    基本运算:
        StrAssign(**s, cstr): 将字符串常量cstr赋值给串s
        DestroyStr(*s): 销毁串
        StrCopy(*s,*t): 复制串，将串t复制给串s
        StrLength(*s): 求串长
        Concat(*s,*t): 串链接，将串s与串t链接起来形成一个新串
        SubStr(*s,i,j): 求子串，返回s中索引i到j组成的字符
        InsStr(*s1, i, *s2): 子串插入，将串s2插入到串s1的i位置
        DelStr(*s,i,j): 子串删除，删除串s中i到j的字符
        RepStr(*s,i,j,*t): 子串替换，将s中i到j位置替换为子串t
        DispStr(*s): 输出串s
}
```
串采用顺序存储时，可以采用紧缩与非紧缩格式，也就是一个字存放一个字符称为非紧缩格式，
否则称为紧缩格式。

**字符串匹配KMP算法**
```c
void GetNext(SqString t, int next[]) {
    int j,k;
    j=0;k=-1;
    next[0] = -1;
    while(j < t.length - 1) {
        if (k==-1 || t.data[j]=t.data[k]) {
            j++;k++;
            next[j] = k;
        }
        else 
            k=next[k];
    }
}

int KMPIndex(SqString s, SqString t) {
    int next[MaxSize], i = 0, j=0;
    GetNext(t, next);
    while(i < s.length && j t.length) {
        if(j==-1||s.data[i] == t.data[j] ) {
            i++;
            j++;
        } else {
            j = next[j];
        }
    }
    if (j >= t.length) {
        return i - t.length;
    } else {
        return -1;
    }
}
```

# 数组
* 一维数组是长度是固定的线性表，数组中的每个数据元素类型相同，结构一致。
* 二维数组可以看做每个元素都是一维数组的一维数组。

**数组的特点**
* 数据元素数目固定。一旦定义了数组，就不再有元素的增减变化。
* 数据元素具有相同的类型
* 数据元素的下标关系具有上下界的约束其下标有序。

## 数组的存储结构
由于数组一般不作为插入和删除运算，也就是，定义了数组，则数组中的数据元素个数和元素之间的关系
就不再发生变动，非常适合顺序结构存储。

二维数据$A\[m\]\[n\]$存储可以采用以行为主或以以列为主两种方法。

设每个元素占用$k$的字节，第一个元素$a_{11}$的地址为$LOC(a_{11}})$。则二维数组
任意元素$a_{ij}$存储地址为:
* 以行为主: $LOC(a_{ij})=LOC(a_{11})+((i-1)\times n + (j-1))\times k$
* 以列为主: $LOC(a_{ij})=LOC(a_{11})+((j-1)\times m + (i-1))\times k$
