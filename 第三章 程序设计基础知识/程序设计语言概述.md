# 一、程序设计语言概述

程序设计语言是为了书写计算机程序而设计的符号语言，用于对计算过程进行描述、组织和推导。

## 1.1 程序设计语言的基本概念

### 1.1.1 低级语言和高级语言
计算机硬件只能识别由0、1字符序列组成的机器指令， **机器指令是最基本的计算机语言** 。机器语言具有如下特点
1. 编制效率低
2. 可读性差
3. 难以理解、修该和维护

为了解决机器指令的缺点，人们设计了汇编语言(**用容易记忆的符号来代替机器指令，来表示机器指令的操作码和操作数**)，例如，ADD表示加法、SUB表示减法等。由于汇编语言是面向机器的语言， **书写格式取决于确定的计算机的机器指令** 。但是汇编语言编写的程序具有较好的可读性，和更高的编程效率。

机器语言和汇编语言被称为低级语言。

为了让程序兼容更多不同的机器，人们设计了 **功能更强、抽象级别更高的语言以支持程序设计这些语言被称为高级语言** 。常见的由Java、C、C++、C#、Python、 PHP等。它们与自然语言比较接近，很大程度上提高了编程效率。

### 1.1.2 编译程序和解释程序

尽管现在借助高级语言与计算机交互，但是计算机只能识别和执行机器语言，所以高级语言设计的程序必须要翻译，而翻译的程序称为“语言处理程序”。程序语言的翻译程序有 **汇编程序、编译程序和解释程序** 

高级语言和汇编语言编程的程序称为源程序，不能直接在计算机上执行。**汇编语言编写的程序需要汇编程序将其翻译为目标程序才能执行；高级语言编写的程序需要对应的编译程序或解释程序翻译后在机器上执行**

* 解释程序又称解释器，它可以直接执行源程序，或者将源程序翻译成某种中间形式再执行。

* 编译程序又称编译器，它首先将源程序翻译成目标程序，将目标程序与库函数链接后形成可执行程序，再在计算机上执行


### 1.1.3 程序设计语言的定义

程序设计语言定义涉及语法、语义和语用3个方面

1. **语法**。是指由程序设计语言基本符号组成程序中的各个语法成分(程序)的一组规则。可以通过形式语言进行描述
    * 词法规则。由基本字符构成的符号(单词)书写规则
    * 语法规则。由符号(单词)构成语法成分的规则

2. **语义**。是程序设计语言中按语法规则构成的各个语法成分的含义。分为静态语义和动态语义。
    * 静态语义。编译时可以确定的语法成分的含义
    * 动态语义。程序执行时才能确定的含义。

3. **语用**。表示构成语言的各个记号和使用者的关系，涉及符号的来源、使用和影响。

语言的实现还涉及语境问题。指理解和实现程序设计语言的环境(包括编译环境和运行环境)

## 1.2 程序设计语言的分类和特点

### 1.2.1 程序设计语言发展概述

程序设计语言发展的根本推动力就是 **对抽象机制的更高要求** ，以及对程序设计活动地支持。就是把机器能够理解的语言提升到机器也能够很好地模仿人类思考问题地形式

* FORTRAN(FORRmula TRANsator)是第一个高级语言，在数值计算领域积累了大量高效而可靠的代码。特性是**接近数学公式的自然描述，很高的执行效率**，广泛应用与并行计算和高性能计算领域

* ALGOL(ALGOrithmic Lanaguage)诞生与晶体管计算机的时代，主导了20世纪60年代程序语言的发展，为软件自动化及软件可靠性的发展奠定基础。采用巴科斯范式BNF表述语言的语法。 **引进了局部性、动态、递归等新概念** 。

* PASCAL语言是一种结构化程序设计语言，是从ALGOL60发展衍生的，但是功能更强且易使用。

* C语言是20世纪70年代发展起来的通用程序设计语言，**兼顾高级语言和汇编语言的特点**，简单、丰富、可移植。UNIX操作系统及其大多数软件都是C编写的。提供了高效的执行语句，允许程序员直接访问操作系统和底层硬件，适用于系统级编程和实时处理应用。

* C++是在C语言基础上于20世纪80年代发展起来的，**与C兼容** 。特点是**增加了类机制**，使其称为面向对象的程序设计语言。提供用户自定义数据结构的现代高级语言特性，**提供了实现基本数据结构和算法的标准库**。代码复用率高。

* Java产生与20世纪90年代，开始是用于开发网络浏览器的小应用程序。保留了C++的基本语法、类和继承等概念，删除C++中不易理解的内容。

* C#是由一种面向对象、运行于`.NET Framework`的高级程序设计语言。

* Object-C继承C的特性，是扩充C的面向对象编程语言。只支持单一父类，不支持多继承

* Ruby一种解释下、面向对象、动态类型的脚本语言。表达了一致性和简单性。

* PHP一种在服务端执行、嵌入HTML文档的脚本语言，风格类似于C语言。能够充分利用服务器性能。

* Python是一种面向对象的解释型程序设计语言，可以用于编写独立的程序、快速脚本和复杂应用的原型。也是一种脚本语言，支持对操作系统的底层访问，也可以将Python源程序翻译为字节码在Python虚拟机上运行。

* JavaScript是脚本语言，广泛应用于web应用的开发，常用来为网页添加动态功能，为用户提供流畅的浏览效果。

* Delphi是一种可视化开发工具，特性是**基于窗体和面向对象的方法、高速的编译器、强大数据库支持、与Windows编程紧密结合以及成熟的组件技术**

* `Visual Basic.NET` 基于`.NET Framework`的面向对象的编程语言。

* 标记语言用一系列约定好的标记来对电子文档进行标记，实现对电子文档的语义结构级格式的定义。

### 1.2.2 程序设计范型

程序设计语言的分类由很多标准。根据程序设计的方法分为**命令式程序设计语言、面向对象程序设计语言、函数式程序设计语言、逻辑型程序设计语言**

1. 命令式程序设计语言(面向过程程序设计语言)：
   
    基于动作的语言，计算被看成动作的序列。程序就是用语言提供的操作命令书写的操作序列。
例如，C、COBOL、ALGOL。编程特点如下
    * 用自顶向下逐步精化的方法编程
    * 按模块组装的方法编程
    * 程序只会包含顺序、分支、循环结构，每种结构只允许单口入单口出。

2. 面向对象程序设计语言

    始于模拟领域发展起来的Simula，在该语言提出了对象和类的概念。C++、Java等都是面向对象的程序设计语言，封装、继承和多态是面向对象编程的基本特征

3. 函数式程序设计语言
   
    类似于$\lambda-$演算为基础的语言。

4. 逻辑型程序设计语言

    形式逻辑为基础的语言。例如PROLOG，是一系列事实、数据对象或事实的具体关系和规则集合。具有很强的推理功能，适用于自动定理证明、专家系统和自然语言处理等问题


## 1.3 程序设计语言的基本成分
程序设计语言的基本成分包括数据、运算、控制和传输等

### 1.3.1 数据成分

程序中数据对象总是对应着应用系统中某些有意义的东西，数据表示则指示了程序中值的组织形式。

**数据类型用于描述数据对象，还用于在基础及其中完成对值的布局，同时还可用于检查表达式中对运算的应用是否正确** 。

数据是程序操作的对象，具有类型、名称、作用域、存储类型和生存期等属性。在程序运行过程中要为它分配内存空间。
*  **名称** ：由用户通过标识符(通常由字母、数字、下划线`_`组成)命名。
*  **类型** ：说明数据占用内存空间的大小和存放形式
*  **作用域** ：说明可以使用数据的代码范围
*  **存储类别** ：说明数据在内存中的位置
*  **生存期** ：说明数据占用内存的时间范围

数据还可进行如下划分
1.  **常量与变量** 。按照程序运行时数据的值能否发生改变进行划分。程序中的数据对象可以具有左值和右值。
    * 左值：值存储单元(地址或容器)
    * 右值：是值(内容)
    
    变量具有左值和右值，程序运行时右值可以改变；常量只有右值，程序运行时不能改变

2.  **全局变量或局部变量** 。按作用域可以将变量分为全局变量和局部变量。一般系统为 **全局变量分配的内存空间在程序运行时不改变** ，而 **局部变量分配的内存空间是动态改变的** 。

3. **数据类型**。按照数据组织形式可以将数据分为基本类型、用户定义类型、构造类型及其他类型。以C/C++为例，数据类型如下
   * 基本类型：整型(int)、字符型(char)、浮点型(float、double)和布尔类型(bool)
    * 特殊类型：空类型(void)
    * 用户定义类型：枚举类型(enum)
    * 构造类型：数组、结构(struct)、联合(union)
    * 指针类型: type *
    * 抽象数据类型：类类型


### 1.3.2 运算成分
程序设计语言的运算成分指明运行使用的运算符号及运算规则。大多数高级语言的基本运算可以分为**算术运算、关系运算、逻辑运算、位运算**等。

**运算符号的使用与数据类型密切相关**。为了由明确唯一的结果，运算符号规定了优先级和结合性
